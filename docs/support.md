# Baby Tracker Support

Last updated: 2026-04-09

## Contact

For support, bug reports, or account questions, open an issue in the GitHub repository:

https://github.com/murphb52/BabyTracker/issues

## Help with common tasks

- Make sure iCloud is enabled on your device if you want sync and caregiver sharing.
- Use the in-app Sync Status screen to confirm whether iCloud is currently available.
- For caregiver sharing, both devices should be signed into iCloud. Full sharing tests work best with two physical devices and two Apple IDs.
- Import and export use local files on your device. Keep exported backups somewhere safe if you want a manual restore option.
